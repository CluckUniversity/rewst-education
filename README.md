# Rewst education

Interactive learning decks for Rewst customer education. Click a link below to open a deck in your browser.

## Decks

| Deck | Link |
|---|---|
| Four dials | [Open](https://cluckuniversity.github.io/rewst-education/decks/four-dials-deck.html) |

## Adding a new deck

1. Drop the HTML file into `/decks/`.
2. Add a row to the table above with the deck name and its link (pattern: `https://cluckuniversity.github.io/rewst-education/decks/FILENAME.html`).
3. Commit and push. GitHub Pages picks up the change automatically (takes a minute or two to go live).

Links open directly in the browser as interactive pages — no download, no GitHub sign-in required.
